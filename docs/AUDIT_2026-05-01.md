## 2. OWASP Top 10 / Security

### P0
- **JSON-LD `</script>` injection across all `<script type="application/ld+json">` blocks** — `src/components/StructuredData.tsx:13`, `src/components/HomepageFAQ.tsx:62`, `src/components/ProductFAQ.tsx:33`, `src/components/Breadcrumbs.tsx:37`, `src/app/shop/[collection]/page.tsx:184,188`, `src/app/shop/clothing/page.tsx:245`, `src/app/shop/spirits/page.tsx:388`, `src/app/field-manual/cocktails/[slug]/page.tsx:269`. All call `JSON.stringify(...)` without escaping `</script>`, `<!--`, `<script`. Inputs include Shopify product titles/descriptions, Sanity content, FAQ metafields. Add helper:
  ```ts
  const safeJsonLd = (data: unknown) => JSON.stringify(data).replace(/</g, '\\u003c')
  ```

### P1
- **PortableText link mark accepts `javascript:` URIs** — `src/components/FieldManualPortableText.tsx:30-39`. Validate `href` against allowlist (`http`, `https`, `mailto`, `tel`, `/`).
- **`/api/referrals/generate` creates Shopify discount codes per email with no email verification** — `src/app/api/referrals/generate/route.ts:26-95`. Require Turnstile; cap codes per day globally; consider only generating from the order webhook.
- **Klaviyo email-based filter assembled with string concat** — `src/app/api/contact/route.ts:192`, `src/app/api/klaviyo-signup/route.ts:119`. Tighten regex to `^[A-Za-z0-9.@_+-]+$`.
- **Sanity GROQ `match $searchTerm` wildcard expansion** — `src/app/api/search/route.ts:125`. Cap query length at 80 chars; strip non-alphanumeric.
- **Trade verify `pin` accepted up to 50 chars** — `src/lib/kv.ts:152-167`. Tighten to `/^\d{4,12}$/`.
- **Server-side Sentry doesn't initialise on Cloudflare Workers** — `instrumentation.ts:2`. Verify `NEXT_RUNTIME` value in deployed env.
- **`?seo_audit=true` query-string bypass of age gate** — `src/components/ClientWrapper.tsx:68`. UK regulators require effective age verification; documented bypass URLs are an audit finding. Sign with token or remove.

### P2
- Klaviyo private key fingerprint in 500 response — `src/app/api/contact/route.ts:118`.
- Webhook handler returns 200 on internal errors — `src/app/api/webhooks/shopify/route.ts:362-366`.
- `_referral_code` from localStorage unbounded — `src/components/CartDrawer.tsx`.
- `/api/recent-orders` iterates all `order:recent:*` KV keys without limit.
- Two Sanity client modules drifting — `src/sanity/client.ts` vs `src/sanity/lib/client.ts`. 13 files import the older one.
